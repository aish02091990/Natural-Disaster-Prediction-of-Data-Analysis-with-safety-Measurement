# Natural-Disaster-Prediction-of-Data-Analysis-with-safety-Measurement
Natural Disaster Prediction of Data Analysis with safety Measurement
Project Overview

Natural disasters such as floods, earthquakes, cyclones, and wildfires cause significant loss of life and property. This project focuses on analyzing historical disaster data and building a machine learning model to predict disaster risk levels and suggest safety measures.

The goal is to help governments, organizations, and individuals take preventive actions and improve disaster preparedness.
